# UI Overview

- Built with Flutter
- Uses Riverpod providers in `ui/lib/core/ppm_providers.dart`
- HTTP client in `ui/lib/core/ppm_client.dart`
- Routes: `/` (Chat), `/settings`

## Settings
- Base URL for Probe API
- Provider + model placeholders (for future chat ingress)
